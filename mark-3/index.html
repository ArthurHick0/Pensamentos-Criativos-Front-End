Eae, aqui é o Hick!


Vou contar um pouco sobre esse projeto, ele é simples porém especial!
Ele foi feito para ser responsivo com base no dispostivo usado para abri-lo. Seu layout é programado para ser flexível, com isso seu menu é alterado conforme o tamanho da tela.
Faça o teste!

Caso use PC: Altere o tamanho da janela do navegador ou entre no modo de inspeção (Aperte a tecla F12 e depois clique no ícone de tablet/celular), dessa forma você poderá alterar as dimensões da janela e o menu irá se moldar ao tamanho.

Caso use dispositivo móvel: Clique nos três pontos na área superior do navegador e ative o modo "Site para computador", após isso deite seu aparelho para centralizar o conteúdo, dessa forma você verá o menu se moldar ao tamanho.
