Eae, aqui é o Hick!

Vou contar um pouco sobre esse projeto, ele é simples porém especial!
Esse site foi feito para ser responsivo com base no dispostivo usado para abri-lo. Seu layout é programado para ser flexível, com isso seu menu é alterado conforme o tamanho da tela.
Faça o teste!

Caso use PC: Altere o tamanho da janela do navegador ou entre no modo de inspeção (Aperte a tecla F12 e depois clique no ícone de tablet/celular), dessa forma você poderá alterar as dimensões da janela e o menu irá se moldar ao tamanho.

Caso use dispositivo móvel: Deite seu aparelho para alterar a orientação do menu e centralizar seu conteúdo. Teste também o layout ativando o modo "Site para computador" nas configurações do seu navegador.
