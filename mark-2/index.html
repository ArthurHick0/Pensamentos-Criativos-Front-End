Introdução sobre cores e RGB

A existência das cores e suas aplicações são baseadas no Círculo Cromático. Esse conceito possui 3 cores que são chamadas de cores "Primárias", isso se deve ao fato de outras cores, chamadas de "Secundárias", surgirem a partir da junção dessas cores primárias. De forma análoga, ao juntar cores secundárias temos a criacão de cores "Terciárias". São elas:

Primárias:
Vermelho
Azul
Amarelo


Secundárias:
Laranja
Violeta/Roxo
Verde

Terciárias:
Amarelo-esverdeado
Amarelo-alaranjado
Vermelho-alaranjado
Vermelho-arroxeado
Azul-arroxeado
Azul-esverdeado

O que foi citado acima se refere a teoria básica sobre as cores e o círculo cromático, porém a aplicação disso na tecnologia funciona diferente. Para telas e monitores é usado o sistema RGB, que tem a mesma lógica anterior só que as cores primárias são: Vermelho, azul e VERDE!

RGB = Red (Vermelho) / Green (Verde) / Blue (Azul)

Cores Complementares

Cores complementares consiste em duas cores que possuem o maior contraste entre si.
A prática resulta em uma cor base e outra que está no ponto oposto a ela no círculo cromático.

Base Vermelha

Este parágrafo representa o vermelho como cor base/primária

Já neste parágrafo temos o ciano/aqua, que é a cor complementar da primária vermelha

Base Azul

Este paráfrago representa o azul como cor base/primária

Já neste parágrafo temos o amarelo, que é a cor complementar da primária azul

Base Amarela

Este parágrafo representa o amarelo como cor base/primária

Já neste parágrafo temos o azul, que é a cor complementar da primária amarela

Cores Análogas

Cores análogas consiste em duas cores consecutivas que são consideradas vizinhas por estarem imediatamente ao lado uma da outra.
A prática resulta em uma cor base e outras duas, que são as mais próximas (Ao lado) dessa cor base, para ambos os lados do círculo cromático.

Base Vermelha

Este parágrafo representa o vermelho como cor base/primária

Neste parágrafo temos o vermelho-arroxeado que é a primeira cor análoga da primária vermelha

Já neste temos o vermelho-alaranjado que é a segunda cor análoga da primária vermelha

Base Azul

Este parágrafo representa o azul como cor base/primária

Neste parágrafo temos o azul-arroxeado que é a primeira cor análoga da primária azul

Já neste temos o azul-esverdeado como segunda cor análoga da primária azul
